Недавно я опубликовал статью о том, как добавить комментарии Вконтакте на сайт с помощью плагина VKontakte API. В этом плагине также можно добавить комментарии Facebook к себе на сайт. Ведь комментарии Вконтакте - это хорошо, а если еще будут Facebook комментарии, то будет еще лучше. Но в той статье я не показал, как это сделать. Так вот теперь я покажу, как это сделать. Принцип тот же самый, что и для Вконтакте. Нужно будет создать приложение в сети Facebook. Нужно будет получить ID этого приложения и ввести его в настройках плагина. Естественно, что вы на этот момент должны быть зарегистрированы в Facebook. Для начала установите плагин VKontakte API. Как это сделать читаем здесь.Открываем настройки плагина.Vkontakte API настройкиелкаем по вкладке Facebook. Настройки Facebook Здесь нужно ввести ID приложения и ID админа. Для этого идем по ссылке, которая отмечена красным на картинке. Создание нового приложения Повторюсь, вы должны быть зарегистрированы в этой сети. На открывшейся странице нужно кликнуть по кнопочке "+Add new app".Веб-сайт Вылезет новое окошко, где вам предложат выбрать платформу вашего приложения. Мы выбираем "Веб-сайт".Название приложения Теперь нам нужно ввести название приложение, которое мы хотим создать. Название нигде публиковаться не будет. Так что сильно заморачиваться с названием не стоит. Называйте приложение так, чтоб вам было понятно, что это за приложение. Ведь у вас может быть несколько созданных приложений. Ну а если у вас уже есть приложение в Face Как только вы начнете писать название снизу откроется кнопка создания приложения. Введите название полностью и нажимайте на эту кнопку. Create app ID Здесь можно выбрать категорию приложения, но она ни на что не влияет. Так что выбираем любую и жмем на кнопку "Create App ID". Настройки приложения Теперь нужно настроить ваше приложение. Кстати, если вы хотите установить комментарии Фейсбука без плагина, то нужно скопировать код на картинке выше и вставить его между тегами и там, где вам нужно. Мы устанавливаем с плагином, поэтому в строке "Site URL" пишем адрес вашего сайта с http протоколом. Если у вас есть мобильная версия сайта можете вписать и этот адрес в поле ниже. Ну а затем "Далее". Так, приложение создано. Сейчас нужно посмотреть его ID.
